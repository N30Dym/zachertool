protected CSVDatenProProtokoll(array<string|int, mixed> $protokolleLayoutsProtokollDaten, string $seperator) : string
  1. Lade aus zachern_protokolllayout.protokoll_layouts die Zeilen, die die ProtokollID aus $protokolleLayoutsProtokollDaten['id'] haben in $protokollLayoutDaten.
  2. Erstelle die Überschriften für diese ProtokollID, die in $csvReturnArray gespeichert werden.
  3. Für jedes Protokoll in $protokollDetailsArray lade die ProtokollDaten mit Protokolldarstellungscontroller::ladeDatenAusDemProtokoll und erstelle daraus einen csv-String, der in $csvReturnArray gespeichert wird.
